Flutter To-Do App with Login/Register

This is a simple Flutter app where you can create an account, login, and manage your to-do list.

Features:

Register: Make a new account with username and password.
Login: Sign in to your account.
Logout: Sign out of the app.

To-Do List:

Add Tasks: Add new tasks with a priority (Low, Medium, High).
Mark Done: Check tasks as done.
Mark All Done: Mark all tasks as done at once.
Delete Task: Remove a task.
Clear All: Remove all tasks.

UI:

Simple and clean design.
Tasks are color-coded by priority:
High => Red
Medium => Orange
Low => Green
